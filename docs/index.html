@magic-modules/git-list

this is the @magic-modules GitList module.shows a list of git repositories with links, gitbadges and descriptions.

installation

npm install magic-modules/module-list

require

first add the component to the assets

// assets/index.jsmodule.exports = {// ..other exportsGitList: require('@magic-modules/module-list'),}

usage

in a page or component

module.exports = () =>  GitList({    header: 'magic',    org: 'magic',    host: 'github', // github is default, gitlab works too.    class: 'custom-class',    items: [      { name: 'core', description: '@magic/core library. grundstein.' },    ],  })
